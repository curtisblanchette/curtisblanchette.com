# fonts/

All brand fonts are self-hosted in this folder. Loaded via `@font-face`
declarations at the top of `../colors_and_type.css`.

- `PPEiko-Light.ttf` (300)
- `PPEiko-Regular.ttf` (400)

- `BasisGrotesquePro-Light.otf` (300) · `-LightItalic`

- `BasisGrotesquePro-Medium.otf` (500) · `-MediumItalic`
- `BasisGrotesquePro-Bold.otf` (700) · `-BoldItalic`
- `BasisGrotesquePro-Black.otf` (900) · `-BlackItalic`
